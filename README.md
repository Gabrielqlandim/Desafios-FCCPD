# Desafios-FCCPD

## Conselhos do desafio
   Para cada desafio é importante derrubar eles após cada teste.

   PARA O DESAFIO 1:
   
     Quando terminar usar o comando:
     
     docker rm -f web client
     
     e depois o comando:
     
     docker network rm desafio1-net

   PARA O DESAFIO 2:
   
     Quando terminar usar o comando:
     
     docker rm -f desafio2-db

   Para O DESAFIO 3:
   
     Quando terminar usar o comando:
     
       docker compose down -v

   PARA O DESAFIO 4:
   
     Quando terminar usar o comando:
     
     docker rm -f service-a service-b
     
     e depois o comando:
     
     docker network rm desafio4-net

   PARA O DESAFIO 5:
   
     Quando terminar usar o comando:
     
       docker compose down
